# Takayasu Arteritis
## Comprehensive Review — Pathophysiology, Diagnosis, Classification & Management 2025

---

## Abstract

Takayasu arteritis (TAK) เป็น large vessel vasculitis ที่พบได้ไม่บ่อยนัก แต่ก่อให้เกิดความทุกข์ทรมานและการเสียชีวิตอย่างมีนัยสำคัญ เนื่องจากการมีส่วนเกี่ยวข้องของ aorta และ major branches ภาวะนี้ส่งผลกระทบต่อผู้ป่วยส่วนใหญ่เป็นผู้หญิงวัยหนุ่มสาว (มักอายุต่ำกว่า 40 ปี) และมีความแตกต่างกันตามภูมิศาสตร์ — พบบ่อยในเอเชียตะวันออกเฉียงใต้ ญี่ปุ่น เม็กซิโก และแอฟริกาใต้

บทความนี้ทบทวนหลักฐานล่าสุดจาก 2021 ACR/Vasculitis Foundation Guideline, 2022 ACR/EULAR Classification Criteria, 2024 Year in Review และ 2025 Nature Reviews Rheumatology เพื่อให้ภาพรวมที่ครบถ้วนของพยาธิสรีรวิทยา การวินิจฉัย การจำแนกประเภท และแนวทางการรักษาที่ปัจจุบัน

**Keywords:** Takayasu arteritis, large vessel vasculitis, ACR/EULAR classification criteria, glucocorticoids, immunosuppression, biologic therapy, aortic stenosis

---

## 1. Introduction

Takayasu arteritis (TAK) เป็น chronic, granulomatous, large-vessel vasculitis ที่มีผลกระทบหลักต่อ aorta และ major branches รวมถึง aortic arch และ major branches (subclavian, carotid, renal arteries) กระบวนการอักเสบทำให้ผนังหลอดเลือดหนาตัวขึ้น ส่งผลให้เกิดการตีบแคบ (stenosis) หรือการขยายตัว (aneurysm) ของหลอดเลือด

**ความสำคัญทางคลินิก:**
- เป็นสาเหตุสำคัญของ stroke และ renal artery stenosis ในผู้ป่วยอายุน้อย
- ความล่าช้าในการวินิจฉัยเฉลี่ย 2-4 ปี เนื่องจากอาการเริ่มต้นไม่จำเพาะ
- มี mortality ที่สูงขึ้นเมื่อเทียบกับประชากรทั่วไป
- การรักษาที่เหมาะสมสามารถชะลอความก้าวหน้าของโรคได้อย่างมีนัยสำคัญ

## 2. Epidemiology

### 2.1 ข้อมูลพื้นฐาน

| Parameter | ข้อมูล |
|-----------|---------|
| **ภูมิภาคที่พบบ่อยที่สุด** | ญี่ปุ่น, เอเชียตะวันออกเฉียงใต้, เม็กซิโก, แอฟริกาใต้, อินเดีย |
| **อัตราส่วนหญิง:ชาย** | 8-9:1 |
| **อายุที่พบบ่อย** | 15-30 ปี (เกือบทั้งหมด <40 ปี) |
| **อุบัติการณ์** | 1-2 ต่อล้านประชากรต่อปี |
| **พยาธิสภาพ** | Granulomatous inflammation ของ arterial wall |

### 2.2 ความแตกต่างตามภูมิศาสตร์

ผู้ป่วยในเอเชียมักมี type V ตาม Numano classification (aortic regurgitation บ่อยกว่า) ในขณะที่ผู้ป่วยในยุโรปและอเมริกาเหนือมักมี type I หรือ II ความแตกต่างนี้อาจสะท้อน genetic susceptibility และ environmental factors ที่ต่างกัน

## 3. Pathophysiology

### 3.1 พยาธิสรีรวิทยา

กระบวนการอักเสบใน Takayasu arteritis ดำเนินผ่านหลายขั้นตอน:

```
Early "Pre-vascular" Phase
     ↓
Granulomatous inflammation ของ vasa vasorum (adventitia)
     ↓
Inflammation แพร่กระจายเข้า media
     ↓
Medial smooth muscle necrosis + fibrosis
     ↓
Neovascularization ผิดปกติใน media
     ↓
Fibrosis ของ arterial wall
     ↓
Stenosis / Aneurysm formation
```

### 3.2 การอักเสบของหลอดเลือด

**ชั้นที่เกี่ยวข้อง:**
- **Adventitia** — บริเวณแรกที่ถูกกระทบ: perivascular granulation tissue, vasa vasorum proliferation
- **Media** — smooth muscle necrosis, elastic fiber destruction
- **Intima** — หนาตัวขึ้นจาก fibrosis, creates stenotic lesions

**Cellular infiltrates:**
- CD4+ T cells (โดดเด่นในระยะแรก)
- CD8+ T cells
- Macrophages
- Multinucleated giant cells (granulomatous inflammation)
- B cells (มีบทบาทน้อยกว่า)

### 3.3 Cytokines และ Signaling Pathways

| Cytokine | บทบาท |
|----------|--------|
| **IL-6** | สูงมากในระยะ active disease — IL-6/IL-6R signaling มีบทบาทหลัก |
| **TNF-α** | กระตุ้น inflammation ใน arterial wall |
| **IFN-γ** | กระตุ้น macrophages, giant cell formation |
| **IL-12, IL-18** | Th1 response |
| **VEGF** | Neovascularization ใน arterial wall |

**โดยเฉพาะ IL-6** มีความสำคัญมาก — ระดับ IL-6 สูงขึ้นในระยะ active disease และสัมพันธ์กับ ESR/CRP ที่สูง

### 3.4 Genetic Associations

**HLA-B*52** เป็น genetic marker ที่แข็งแกร่งที่สุดสำหรับ Takayasu arteritis:
- พบมากในผู้ป่วยญี่ปุ่น (40-60%)
- สัมพันธ์กับ IL12B polymorphism ด้วย
- อยู่ในกลุ่ม "MHC-I-opathies" ร่วมกับ Behçet's disease และ psoriasis

**Other susceptibility loci:**
- IL12B, IL6R
- FCGR2A
- PTPN22
- และยีนที่เกี่ยวกับ TGF-β signaling

## 4. Clinical Presentation

### 4.1 อาการทางคลินิก — ระยะ Early (Pre-vascular) Phase

ในระยะแรก อาการมักไม่จำเพาะ ทำให้การวินิจฉัยล่าช้า:

| อาการ | ความถี่ |
|-------|---------|
| **Constitutional symptoms** | 50-80% — ไข้, เหนื่อยล้า, น้ำหนักลด, เบื่ออาหาร |
| **Arthralgia / Myalgia** | 30-50% |
| **อาการทางผิวหนัง** | ผื่น, erythema nodosum (บางส่วน) |
| **ไข้ไม่ทราบสาเหตุ** | พบได้บ่อย — อาจเป็นอาการนำหลายเดือนก่อนมี vascular symptoms |

### 4.2 อาการทางคลินิก — ระยะ Late (Vascular) Phase

เมื่อ arterial wall ถูกทำลายมากขึ้น อาการจำเพาะมากขึ้น:

**Vascular insufficiency จาก arterial stenosis/occlusion:**

| อาการ | สาเหตุ |
|-------|---------|
| **Claudication** | Reduced blood flow ไปยังแขน/ขา |
| **Absence/weakness ของ peripheral pulses** | Subclavian หรือ brachiocephalic artery stenosis |
| **Blood pressure discrepancy** | แขนข้างละ >10 mmHg |
| **Carotidynia** | ความเจ็บบริเวณ carotid arteries |
| **Dizziness, syncope** | Carotid/vertebral artery stenosis |
| **Visual disturbances** | Ophthalmic artery involvement |
| **Stroke, TIA** | Extracranial carotid/vertebral involvement |

**จุดเด่นที่ช่วยวินิจฉัยทางคลินิก:**
- **Supraclavicular/subclavian bruit** — พบได้บ่อย
- **Absence of radial pulse** (辶脈 Missing pulse)
- **Blood pressure ไม่เท่ากันระหว่างสองแขน** >10 mmHg

**Aortic involvement:**

| อาการ | สาเหตุ |
|-------|---------|
| **Aortic regurgitation / Heart failure** | Aortic root dilation, type III dissection |
| **Hypertension จาก renal artery stenosis** | พบได้บ่อยมาก (30-60%) |
| **Chest pain** | aortic involvement, coronary artery involvement |
| **Abdominal pain** | mesenteric artery involvement |

**Cardiac manifestations:**
- Aortic regurgitation (โดยเฉพาะ type V — aortic arch + descending aorta)
- Left ventricular dysfunction
- Coronary artery involvement (10-15%)
- Cardiomyopathy (rare)

### 4.3 Numano Classification

| Type | ตำแหน่งที่เกี่ยวข้อง |
|------|---------------------|
| **Type I** | Aortic arch + branches |
| **Type IIa** | Ascending aorta + aortic arch + branches |
| **Type IIb** | Ascending aorta + aortic arch + branches + thoracic descending aorta |
| **Type III** | Thoracic descending aorta ± abdominal aorta ± renal arteries |
| **Type IV** | Abdominal aorta ± renal arteries |
| **Type V** | Combined type IIb + IV (entire aorta) |

## 5. ACR/EULAR 2022 Classification Criteria

### 5.1 Absolute Requirements

```
ก่อนใช้ criteria ต้องมี:
1. Age ≤ 60 ปี ณ วันวินิจฉัย
2. มี evidence ของ vasculitis จาก imaging หรือ pathology
```

### 5.2 Classification Criteria — Scoring System

รวม ≥5 คะแนน จึงจัดเป็น Takayasu arteritis:

| หมวด | รายการ | คะแนน |
|------|--------|-------|
| **Clinical criteria** |  |  |
|  | Female sex | +1 |
|  | Angina pectoris / Cardiac ischemia | +2 |
|  | Carotid artery tenderness | +1 |
|  | Reduced brachial artery pulse | +2 |
|  | Blood pressure discrepancy between arms >10 mmHg | +1 |
|  | Claudication (extremities) | +1 |
|  | Systolic BP in right arm <90 mmHg / Undetectable | +2 |
| **Imaging criteria** |  |  |
|  | Number of affected aortic territories: |  |
|  | — 1 territory | +1 |
|  | — 2 territories | +2 |
|  | — 3 or more territories | +3 |
|  | **โดยรวม** max score = **10** |

### 5.3 หมายเหตุสำคัญ

- หากมี PET/CT, CTA, MRA หรือ Doppler ultrasound แสดง circumferential wall thickening หรือ vascular FDG uptake ที่สอดคล้องกับ vasculitis ถือว่ามี "evidence of vasculitis"
- ต้องระวังแยกจาก atherosclerosis ที่พบบ่อยในผู้ป่วยสูงอายุ
- ผู้ป่วยที่มี aortic dissection อย่างเดียว (ไม่มี vasculitic changes) ไม่ควรจัดเป็น TAK

## 6. Investigations

### 6.1 Laboratory Tests

| การตรวจ | ข้อมูลที่ได้ |
|---------|-------------|
| **ESR** | สูงในระยะ active แต่ไม่จำเพาะ |
| **CRP** | สูงในระยะ active — IL-6 driven |
| **IL-6** | สูงใน active disease — มีประโยชน์ติดตามตอบสนองต่อ tocilizumab |
| **Anemia of chronic disease** | พบได้บ่อย |
| **Serum creatinine** | ติดตาม renal artery stenosis |
| **Autoantibodies** | RF, ANA มักลบ — ช่วยแยกจาก SLE |

**ข้อจำกัด:** ESR/CRP อาจปกติในผู้ป่วย 30-40% แม้ในระยะ active — ไม่ควรใช้ตัดสินความดีอกของโรคเพียงอย่างเดียว

### 6.2 Imaging

Imaging เป็น cornerstone ของการวินิจฉัยและติดตามโรค:

| Imaging | ข้อดี | ข้อจำกัด |
|---------|-------|----------|
| **Catheter angiography (DSA)** | Gold standard สำหรับ stenosis — ใช้ตัดสินว่าต้องทำ intervention | Invasive, radiation, contrast nephropathy |
| **MRA (Magnetic Resonance Angiography)** | Non-invasive, ไม่มี radiation, ดู wall thickening ได้ | ราคาแพง, CI กับ pacemaker/ICD |
| **CTA (CT Angiography)** | เร็ว, หาง่าย, ดู wall calcification ได้ดี | Radiation, contrast nephropathy |
| **PET/CT** | ดีมากสำหรับ active inflammation — FDG uptake | ราคาแพง, radiation |
| **Ultrasound (Color Doppler)** | Non-invasive, ไม่มี radiation, ดู wall edema ระยะแรก | Operator-dependent, ดู aorta ได้ไม่ดี |

### 6.3 FDG-PET/CT — บทบาทพิเศษ

FDG-PET/CT มีความสำคัญเฉพาะใน Takayasu arteritis:

- **Active arterial wall inflammation** แสดงเป็น increased FDG uptake
- **ติดตาม active disease** ได้ดีกว่า ESR/CRP ในหลายกรณี
- **วางแผนการรักษา** — ช่วยระบุว่าผู้ป่วยยัง active หรือไม่ก่อนจะลดยา
- **ตรวจสอบ response ต่อ tocilizumab** — IL-6 level ลดลงเมื่อ disease controlled

### 6.4 Histopathology

**Granulomatous inflammation กับ multinucleated giant cells** เป็นลักษณะเฉพาะ:

```
Microscopic findings:
- Granulomatous inflammation in media and adventitia
- Multinucleated giant cells (Langhans type)
- Fragmentation of elastic fibers
- Neovascularization
- Fibrosis (late stage)
- Intimal hyperplasia
```

**เมื่อไหร่ควรทำ biopsy?**
- มักไม่จำเป็นถ้า imaging ชัดเจน
- ทำได้จาก subclavian/axillary artery หรือ temporal artery (rarely involved)
- มีประโยชน์เมื่อ differential diagnosis รวมถึง atherosclerosis, giant cell arteritis, or malignancy

## 7. Treatment

### 7.1 ACR/Vasculitis Foundation 2021 Guidelines — Key Recommendations

**Overall principle:** ไม่มี strong recommendations มากเนื่องจาก evidence ส่วนใหญ่มาจาก observational studies

#### Initial Therapy

| สถานการณ์ | แนะนำ |
|-----------|-------|
| **Active disease, no major complications** | Glucocorticoids เป็น first-line + glucocorticoid-sparing agent |
| **Active disease + progressive vascular lesions** | Glucocorticoids + immunosuppressant (non-glucocorticoid) |
| **Refractory disease** | พิจารณา biologic agents (TNF-α inhibitors, tocilizumab) |

#### Glucocorticoids

**Prednisone regimen ที่แนะนำ:**
- เริ่มต้น: Prednisone 0.5 mg/kg/day (หรือ equivalent)
- ลด dose ค่อยเป็นค่อยไปตาม clinical response
- ลดให้เหลือ ≤10 mg/day ภายใน 3-6 เดือน
- ควรใช้ bone protection (bisphosphonate, calcium, vitamin D)

**หลีกเลี่ยง rapid taper** — เพิ่มความเสี่ยง relapse

#### Glucocorticoid-Sparing Agents

| ยา | ข้อมูล |
|----|-------|
| **Methotrexate** | ที่นิยมที่สุด, มีหลักฐานมากที่สุด |
| **Azathioprine** | ใช้ได้ แต่หลักฐานน้อยกว่า |
| **Mycophenolate mofetil** | ทางเลือก แต่ข้อมูลจำกัด |
| **Leflunomide** | มีบ้างใน literature |

#### Biologic Agents

| ยา | สถานะ | หมายเหตุ |
|----|-------|---------|
| **TNF-α inhibitors (adalimumab, infliximab, etanercept)** | มีหลักฐานมาก, ใช้ใน refractory | อาจเพิ่มความเสี่ยงติดเชื้อ |
| **Tocilizumab (anti-IL-6R)** | มีหลักฐานเพิ่มขึ้นเรื่อยๆ | ดีสำหรับผู้ป่วยที่ IL-6 สูง |
| **Abatacept (CTLA4-Ig)** | มีบ้างใน refractory | หลักฐานยังจำกัด |
| **Rituximab** | มีบ้าง | ไม่แนะนำเป็น first-line |

#### Surgical/Endovascular Intervention

**เมื่อไหร่ควรทำ:**
- Critical limb ischemia ที่ไม่ตอบสนองต่อยา
- Severe renal artery stenosis ที่มี refractory hypertension
- Severe aortic regurgitation
- Significant carotid stenosis (>70%) ที่มี TIA/stroke symptoms

**ข้อควรระวัง:**
- **ทำเมื่อโรคสงบ (quiescent)** — active inflammation ทำให้ graft/stent ล้มเหลวเร็วกว่า
- ควรใช้ glucocorticoids ก่อนทำ 2-4 สัปดาห์
- Percutaneous transluminal angioplasty (PTA) กับ bare-metal stent แนะนำมากกว่า drug-eluting stents

## 8. Monitoring & Outcomes

### 8.1 การติดตาม Disease Activity

| วิธี | ความถี่ | ข้อหมายเหตุ |
|------|--------|-------------|
| **Clinical assessment** | ทุก 1-3 เดือน | อาการ, pulse exam, BP |
| **ESR/CRP** | ทุก 1-3 เดือน | ไม่จำเพาะ — 30-40% active แม้ปกติ |
| **IL-6** | ถ้าใช้ tocilizumab | ดีกว่า ESR/CRP สำหรับ monitor |
| **Imaging** | ทุก 6-12 เดือน | MRA หรือ CTA |
| **FDG-PET/CT** | เมื่อสงสัย active disease | ดีสำหรับ detect subclinical inflammation |

### 8.2 Relapse Rate

- Relapse rate สูงมาก: 30-70% ในช่วง 5 ปี
- Relapse มักเกิดเมื่อลด glucocorticoid dose
- ปัจจัยเสี่ยงต่อ relapse: high baseline disease activity, inadequate initial therapy

### 8.3 Prognosis

**Mortality:**
- 5-year survival: 80-95% ในประเทศพัฒนาแล้ว
- สาเหตุการตายหลัก: stroke, myocardial infarction, heart failure, aortic rupture

**Functional outcomes:**
- ผู้ป่วยจำนวนมากมี disability จาก claudication หรือ visual loss
- Quality of life ลดลงอย่างมีนัยสำคัญในผู้ป่วยที่มี active disease ซ้ำๆ

## 9. Summary & Key Take-home Points

| ข้อ | Key Message |
|-----|-------------|
| **1** | Takayasu arteritis ควรนึกถึงในผู้หญิงอายุ <40 ปีที่มี pulse abnormalities, claudication, หรือ hypertension ที่ไม่สามารถอธิบายได้ |
| **2** | Constitutional symptoms (ไข้, เหนื่อยล้า, น้ำหนักลด) มักเป็นอาการนำ — วินิจฉัยล่าช้าเฉลี่ย 2-4 ปี |
| **3** | Imaging (MRA, CTA, PET/CT) เป็น cornerstone ของการวินิจฉัย — ต้องมี evidence ของ vasculitis ก่อนจะวินิจฉัย |
| **4** | ACR/EULAR 2022 criteria ต้องการ age ≤60 + evidence of vasculitis + ≥5 คะแนนจาก clinical และ imaging criteria |
| **5** | Glucocorticoids เป็น first-line treatment — แต่ควรใช้ร่วมกับ glucocorticoid-sparing agent เพื่อลด dose เร็วที่สุด |
| **6** | TNF-α inhibitors และ tocilizumab มีประสิทธิภาพดีใน refractory disease — tocilizumab ดีเป็นพิเศษเมื่อ IL-6 สูง |
| **7** | ESR/CRP ไม่จำเพาะ — 30-40% active แม้ ESR/CRP ปกติ ควรใช้ imaging ช่วยตัดสิน |
| **8** | Surgical intervention ควรทำเมื่อโรคสงบ (quiescent) — active inflammation ทำให้ล้มเหลวเร็ว |
| **9** | Relapse rate สูงมาก (30-70%) — ต้องติดตาม密切และลดยาอย่างช้าๆ |
| **10** | HLA-B*52 สัมพันธ์กับโรคอย่างแข็งแกร่ง — อยู่ในกลุ่ม MHC-I-opathies ร่วมกับ Behçet's และ psoriasis |
