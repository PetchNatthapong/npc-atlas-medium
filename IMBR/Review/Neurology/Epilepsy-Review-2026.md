## Abstract

Epilepsy is one of the most common neurological disorders, affecting approximately 50 million people worldwide. This review summarizes recent advances in classification, diagnosis, and treatment based on 2024-2025 ILAE guidelines and major clinical trials.

---

## 1. Definition & Classification

### 1.1 Seizure Classification (ILAE 2017)

| Type | Characteristics |
|------|-----------------|
| **Focal Onset** | Starts in one hemisphere |
| → Aware vs Impaired Awareness | |
| → Motor vs Non-motor | |
| **Generalized Onset** | Both hemispheres from start |
| → Tonic-clonic, Absence, Myoclonic, Atonic | |
| **Unknown Onset** | Unclassified |

### 1.2 Epilepsy Classification

| Type | Features |
|------|----------|
| **Focal Epilepsy** | Single or multiple foci |
| **Generalized Epilepsy** | Generalized spike-wave |
| **Combined Generalized + Focal** | Dravet, LGS |
| **Unknown** | Cannot determine |

## 2. Diagnosis

### 2.1 Clinical Diagnosis

**Key History Questions:**
1. Witnessed seizure? (Jerking, stiffening, staring)
2. Prodrome? (Aura)
3. Tongue biting, incontinence?
4. Post-ictal confusion?
5. Sleep-related?

### 2.2 EEG Findings

| Pattern | Clinical Significance |
|---------|---------------------|
| **Interictal epileptiform discharges** | Sharp waves, spikes | ✓ Diagnostic |
| **3 Hz spike-and-wave** | Childhood absence | ✓ |
| **Focal slowing** | Underlying lesion | ✓ |
| **Photoparoxysmal** | Photosensitive epilepsy | ✓ |

### 2.3 Neuroimaging

**MRI Epilepsy Protocol:**
- High-resolution T2/FLAIR
- Coronal hippocampus
- Thin-slice whole brain

**Findings that suggest focal epilepsy:**
- Mesial temporal sclerosis
- Cortical dysplasia
- Tumor (Ganglioglioma, DNET)
- Hippocampal atrophy

## 3. Antiseizure Medications (ASMs)

### First-line Treatments

| Drug | Indications | Dose | Side Effects |
|------|-------------|------|-------------|
| **Levetiracetam** | Focal, Generalized | 1000-3000 mg/day | Behavioral changes |
| **Lamotrigine** | Focal (women) | 200-400 mg/day | Rash (SJS) |
| **Valproate** | Generalized | 1000-2000 mg/day | Weight, tremor, hepatotoxicity |
| **Carbamazepine** | Focal only | 600-1200 mg/day | Hyponatremia, rash |
| **Oxcarbazepine** | Focal | 900-1800 mg/day | Hyponatremia |
| **Lacosamide** | Focal | 200-400 mg/day | Dizziness |

### ASM Selection by Epilepsy Type

| Epilepsy Type | First-line | Alternatives |
|---------------|-------------|---------------|
| **Focal** | Levetiracetam, Lamotrigine | Lacosamide, Carbamazepine |
| **Generalized Tonic-Clonic** | Valproate, Levetiracetam | Lamotrigine |
| **Absence** | Ethosuximide, Valproate | Lamotrigine |
| **Myoclonic** | Valproate, Levetiracetam | Clonazepam |
| **Dravet Syndrome** | Stiripentol + Valproate + Clobazam | Cannabidiol |

## 4. Treatment Algorithms

### 4.1 Newly Diagnosed Focal Epilepsy

```
Step 1: Levetiracetam หรือ Lamotrigine
    ↓ (Seizure freedom > 1 year)
Step 2: Continue same drug
    
    ↓ (Seizure not controlled)
Step 3: Switch to alternative first-line
    OR Add second drug
    
    ↓ (2 drugs failed)
Step 4: Consider pre-surgical evaluation
```

### 4.2 Refractory Epilepsy Definition
- **Failed 2 appropriate ASMs**
- Consider epilepsy surgery evaluation

## 5. Status Epilepticus

### 5.1 Definitions

| Type | Duration |
|------|----------|
| **Convulsive SE** | >5 min |
| **Non-convulsive SE** | >10 min |
| **Refractory SE** | Failed 2 lines of treatment |

### 5.2 Treatment Protocol

| Time | Intervention |
|------|--------------|
| **0-5 min** | ABCs, IV Lorazepam 0.1 mg/kg |
| **5-20 min** | Repeat Lorazepam OR Midazolam 10 mg IM |
| **20-40 min** | Fosphenytoin 20 mg PE/kg OR Valproate 40 mg/kg |
| **>40 min** | Phenobarbital OR Consider anesthesia |

## 6. Special Situations

### 6.1 Women with Epilepsy (WWE)

**Pregnancy:**
- Plan pregnancy if possible
- Continue ASMs during pregnancy
- Folic acid 4 mg/day
- High-risk obstetric care

**Safe ASMs in pregnancy:**
- Lamotrigine ✓
- Levetiracetam ✓
- Carbamazepine ✓ (some risk)

**Avoid in pregnancy:**
- Valproate ❌ (NTD risk)
- Carbamazepine ❌ (minor NTD risk)

### 6.2 Elderly

- Start low, go slow
- Avoid enzyme-inducing drugs (CBZ, PHT)
- Levetiracetam, Lamotrigine preferred

### 6.3 Seizure Clusters

**Acute Repetitive Seizures:**
- buccal/IM midazolam
- Rectal diazepam
- Nasal diazepam (Stesold)

## 7. Surgical Evaluation

### 7.1 When to Consider

| Indication | Evaluation |
|-----------|------------|
| **Focal epilepsy** | MRI, EEG, PET, SPECT |
| **Mesial temporal sclerosis** | Excellent surgical outcome |
| **Cortical dysplasia** | 50-70% seizure-free |

### 7.2 Presurgical Tests

1. **Video-EEG Monitoring** — ictal onset zone
2. **MRI Epilepsy Protocol** — structural lesion
3. **PET Scan** — hypometabolism
4. **MEG/EEG** — source localization
5. **Intracranial EEG** — if non-invasive insufficient

## 8. Emerging Therapies

### 8.1 Cannabinoids
- **CBD (Epidiolex):** Dravet, LGS, TSC
- 99% CBD, minimal THC
- Drug interactions (CYP3A4, 2C19)

### 8.2 New ASMs (2022-2025)

|------|-----------|-------------|

| **Ganaxolone** | GABA-A mod | CDKL5 deficiency |
| **Soticlestat** | Cholesterol 24-hydroxylase inhibitor | Dravet, LGS |

### 8.3 Devices

- **Responsive Neurostimulation (RNS)**

## 9. Summary Points

### Key Takeaways

1. **Classification** — Focal vs Generalized guides treatment
2. **Diagnosis** — Clinical + EEG + MRI
3. **ASM Selection** — Depends on seizure type and patient factors
4. **Refractory** — 2 drug failures → consider surgery
5. **Women** — Plan pregnancy, avoid valproate
6. **New Therapies** — Cannabinoids, new ASMs, devices
